# 智能起名小程序 - 开发计划

## 一、项目概述

**项目名称**：智能起名（暂定）
**类型**：微信小程序
**核心功能**：根据用户输入的姓氏、生日、性别等信息，结合传统起名文化（八字五行、生肖、诗经成语等），智能生成寓意美好的名字
**目标用户**：准父母（新生儿起名）、成人（改名需求）

---

## 三、技术架构

### 前端
- **框架**：微信小程序原生开发 或 uni-app（跨平台）
- **UI库**：Vant Weapp / 自定义组件
- **状态管理**：MobX / globalData

### 后端
- **语言**：Node.js / Python
- **框架**：Express / FastAPI
- **数据库**：MySQL（名字库）+ Redis（缓存）

### 部署
- **服务器**：腾讯云轻量服务器 / Vercel（前端）
- **域名**：已完成备案的域名

### 起名逻辑选择（选一个或组合）

```
A. 八字五行起名
   - 需要提供出生精确时间（时）
   - 根据八字喜用神起名
   - 补足五行缺失

B. 生肖属相起名
   - 根据属相宜用字
   - 避开相冲相克字

C. 诗经成语起名
   - 从《诗经》《楚辞》、唐诗宋词、成语中提取
   - 有出处、有寓意

D. 随机智能推荐
   - 结合音形义评分
   - 随机生成寓意好的名字
```

### UI风格选择（选一个）

A. 古风传统
   - 配色：红、金、黑、白

B. 清新现代
   - 配色：绿、蓝、白
   - 元素：简约、扁平插画

C. 可爱卡通
   - 配色：粉、黄、紫
